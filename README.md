# Exploratory-Data-Analysis-EDA-Project

## Project Objective
Analyze student performance data to identify trends, correlations, and key influencing factors using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Statistical Summary
- Correlation Analysis
- Heatmap Visualization
- Gender-wise Analysis
- Lunch Type Analysis
- Pair Plot Visualization

## Key Insights
- Reading and writing scores are strongly correlated.
- Female students perform better in reading and writing.
- Standard lunch improves student performance.
- Higher parental education positively impacts scores.

## Project Outcome
Developed analytical thinking and data exploration skills through visualization and statistical analysis.
